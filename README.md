# Computational Imaging & Machine Learning Portfolio
Earl Wong

This repository contains a structured technical portfolio covering computational imaging, machine learning, and real-time vision systems.

The material was developed to formalize first-principles understanding of camera systems and modern machine learning methods used in visual perception and video processing.

My professional work focuses on designing and shipping end-to-end camera and vision features.   

This portfolio complements that work by documenting the underlying system architecture, algorithms, and ML techniques involved.

# Repository Structure

## Vision / Computational Imaging

System-level breakdown of modern camera pipelines including:

• Real-time video architecture 
• Latency and bandwidth  
• Optics and image sensor fundamentals  
• ISP pipelines  
• GPU and NPU acceleration strategies  
• Autofocus and color constancy algorithms  
• Classical and ML-based noise reduction  
• Video-specific temporal processing  
• Vision evaluation metrics  

These materials reflect the end-to-end thinking required to build production camera systems.
